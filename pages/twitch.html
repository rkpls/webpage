Twitch-Befehle

Zuschauer

/me [Nachricht] Hebt die Nachricht hervor, indem sie im Farbton deines Benutzernamens erscheint.
/w [Benutzername] [Nachricht] Sendet eine private Nachricht an einen Benutzer.
/mods Zeigt eine Liste der aktuellen Moderatoren im Kanal an.
/vips Zeigt die VIPs des Kanals an.
/help Zeigt eine Liste der verfügbaren Chat-Befehle an.
/marker [Beschreibung] Setzt eine Markierung im VOD mit optionaler Beschreibung (Streamer/Editor).

Moderator

/ban [Benutzername] [Grund] Bannt einen Benutzer permanent.
/unban [Benutzername] Hebt einen Ban auf.
/timeout [Benutzername] [Sekunden] Bannt einen Benutzer temporär.
/untimeout [Benutzername] Hebt den Timeout eines Benutzers auf.
/clear Löscht den gesamten Chatverlauf.
/slow [Sekunden] Aktiviert den Slow-Modus (Limitierung der Nachrichten).

Nightbot-Befehle

Zuschauer

!commands Zeigt eine Liste der verfügbaren Befehle.
!uptime Zeigt die aktuelle Stream-Laufzeit.
!title Zeigt den aktuellen Stream-Titel an.
!game Zeigt das aktuell gestreamte Spiel an.
!lurk Informiert, dass du im „Lurk“-Modus bist.
!watchtime Zeigt, wie lange ein Benutzer den Stream gesehen hat.

Moderator

!addcom [Befehl] [Nachricht] Erstellt einen benutzerdefinierten Befehl.
!editcom [Befehl] [Nachricht] Bearbeitet einen bestehenden Befehl.
!delcom [Befehl] Löscht einen benutzerdefinierten Befehl.
!game [Spiel] Ändert das aktuelle Spiel im Stream.
!title [Titel] Ändert den Stream-Titel.
!winner Wählt einen zufälligen Gewinner aus dem Chat.
!poll Startet eine Umfrage.
