Twitch Befehle

Twitch-Befehle


Zuschauer +

/help Zeigt eine Liste der verfügbaren Chat-Befehle an.
/me [Nachricht] Hebt die Nachricht hervor, indem sie im Farbton deines Benutzernamens erscheint.
/w [Benutzername] [Nachricht] Sendet eine private Nachricht an einen Benutzer.
/vote [index] Stimmt ab. Index ist die Abstimmungsoption.
/gift [Anzahl] Verschenkt eine Anzahl Gifted-Subs.
/mods Zeigt eine Liste der aktuellen Moderatoren im Kanal an.
/vips Zeigt die VIPs des Kanals an.
/block [Nutzername] Blockt einen Nutzer.
/unblock [Nutzername] Entblockt einen Nutzer.

Moderator +

/announce [Nachricht] Hebt die Nachricht hervor. Farboption /announceblue -green -orange -purple
/pin [Nachricht] Eine Nachricht anpinnen
/shoutout [Benutzername] Shoutout an einen Streamer. Generiert Follow Option, angepinnt im Chat.
/warn [Benutzername] [Grund] Einen Nutzer an die Chatregeln erinnern.
/monitor [Benutzername] Die Nachrichten eines Nutzers beobachten.
/unmonitor [Benutzername] Die Nachrichten eine Nutzers nicht mehr beobachten.
/restrict [Benutzername] Die Nachrichten eines Nutzers einschränken.
/unrestrict [Benutzername] Die Nachrichten eine Nutzers nicht mehr einschränken.
/ban [Benutzername] [Grund] Bannt einen Benutzer permanent.
/unban [Benutzername] Hebt einen Ban auf.
/timeout [Benutzername] [Sekunden] Bannt einen Benutzer temporär.
/untimeout [Benutzername] Hebt den Timeout eines Benutzers auf.
/clear Löscht den gesamten Chatverlauf.
/uniquechat Aktiviert den Slow-Modus (Limitierung der Nachrichten).
/uniquechatoff Deaktiviert den Slow-Modus.
/slow [Sekunden] Aktiviert den Slow-Modus (Limitierung der Nachrichten).
/slowoff Deaktiviert den Slow-Modus.
/followers [Zeit] Aktiviert den Follower-Modus (nur Follower mit einer bestimmten Mindestzeit können schreiben).
/followersoff Deaktiviert den Follower-Modus.
/subscribers Aktiviert den Abonnenten-Modus (nur Subs können schreiben).
/subscribersoff Deaktiviert den Abonnenten-Modus.
/shield Aktiviert den Schutzmodus, der automatisch strengere Regeln aufstellt.
/shieldoff Deaktiviert den Schutzmodus.
/raid [Kanalname] Startet einen Raid zu einem anderen Kanal.
/unraid Hebt einen aktiven Raid auf.
/host [Kanalname] Hostet einen anderen Kanal.
/unhost Beendet das Hosting.

Streamer

/commercial [Dauer in Sekunden] Startet eine Werbepause (30, 60, 90, 120, 150 oder 180 Sekunden).
/announce [Nachricht] Hebt eine Nachricht für alle im Chat hervor.
/marker [Beschreibung] Setzt eine Markierung im VOD mit optionaler Beschreibung.

Nightbot-Befehle


Zuschauer +

!commands Zeigt eine Liste der verfügbaren Befehle.

Kanalabhängig, Meistverwendet:

!uptime Zeigt die aktuelle Stream-Laufzeit.
!title Zeigt den aktuellen Stream-Titel an.
!game Zeigt das aktuell gestreamte Spiel an.
!lurk Informiert, dass du im „Lurk“-Modus bist.
!unlurk Informiert, dass du nicht mehr im „Lurk“-Modus bist.


Moderator +

!commands add [Befehl] [Nachricht] Erstellt einen benutzerdefinierten Befehl.
!commands edit [Befehl] [Nachricht] Bearbeitet einen bestehenden Befehl.
!commands delete [Befehl] Löscht einen benutzerdefinierten Befehl.
!game [Spiel] Ändert das aktuell gestreamte Spiel im Stream.
!title [Titel] Ändert den Stream-Titel.
!poll new [Frage] | [Option 1] | [Option 2] ... Startet eine neue Umfrage mit mehreren Antwortmöglichkeiten.
!poll end Beendet die aktuelle Umfrage und zeigt die Ergebnisse an.
